### 새 저장소 만들기

> 1.	**git init**
> 2.	**git remote add origin 원격 서버 주소**

원하는 폴더에서 **git init** 로 사전 준비

원격 서버 주소에 링크

### 새 내용 업데이트 하기

> 1.	\**git add 파일 이름 or git add *\*\*
> 2.	**git commit -m "설명 메시지"**
> 3.	**git push origin master**

### git 원격저장소에 있는 내용 로컬에 합치기

> 2.	**git merge origin master**
